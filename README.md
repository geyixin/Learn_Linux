## 目录
* cd /home 进入 '/ home' 目录' 
* cd .. 返回上一级目录 
* cd ../.. 返回上两级目录 
* cd 进入个人的主目录 
* cd ~user1 进入个人的主目录 
* cd - 返回上次所在的目录 
* pwd 显示工作路径 
* mkdir dir1 创建一个叫做 'dir1' 的目录' 
* rm -rf dir1 删除一个叫做 'dir1' 的目录并同时删除其内容 
* mv dir1 new_dir 重命名/移动 一个目录 
## 文件
* ls 查看目录中的文件 
* ls -F 查看目录中的文件 
* ls -l 显示文件和目录的详细资料 
* ls -a 显示隐藏文件 
* rm -f file1 删除一个叫做 'file1' 的文件' 
## 系统
* shutdown -h now 关闭系统
* shutdown -r now 重启
* reboot 重启
